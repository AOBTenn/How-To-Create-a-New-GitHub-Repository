<h2>Environments and Technologies Used</h2>

- Desktop Pc
- Internet

- Windows 11 Home</b>

<h2>List of Prerequisites</h2>

- Laptop or Desktop Pc                                                                                                                                 
- Operating system on said laptop or desktop
- Wifi Access
- GitHub Account
